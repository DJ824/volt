Thread Pool library in Modern C++ 

## ThreadPool API

Include the header and create a pool with the number of worker threads you want:

```cpp
#include "thread_pool.h"

ThreadPool pool{std::thread::hardware_concurrency()};
pool.start();
```

Submit work that returns a `std::future`:

```cpp
auto value = pool.submit([] {
    return 42;
});

int result = value.get();
```

Submit fire-and-forget work:

```cpp
pool.submit_detached([] {
    do_work();
});

pool.wait_for_tasks();
```

Tasks can spawn more tasks by using the context variants:

```cpp
pool.submit_detached_ctx([](ThreadPool::TaskContext& ctx) {
    ctx.spawn_detached([] {
        do_child_work();
    });

    auto child = ctx.spawn_returning([] {
        return compute_value();
    });
});
```

Lifecycle:

```cpp
pool.wait_for_tasks(); // wait until pending work reaches zero
pool.stop();           // request worker shutdown
pool.join();           // join worker threads
```
